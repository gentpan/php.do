php.do（LiteBBS）— 轻量 PHP 论坛
线上：https://php.do

部署环境：FrankenPHP + PHP 8.5 + MySQL + Caddy
完整说明请阅读 README.md（含 Caddyfile 示例、Git 推送部署、命名约定）。

环境要求：PHP 8.5+、mysqli/json 扩展、MySQL 8.x/9.x。
安装：复制 config.example.php 为 config.php → 访问 install/install.php。
升级：访问 install/upgrade.php 后删除。

默认管理员 admin / admin123，登录后请立即改密。
