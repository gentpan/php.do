php.do

环境要求：
PHP 8.5+
MySQL 8.x 或 MySQL 9.x beta
必须开启 mysqli、json 扩展
建议开启 mbstring 扩展；使用 S3/R2 上传时需要 curl 扩展

不再支持 PHP 8.4 及以下版本，不支持 MySQL 5.7 / MariaDB。安装和运行时都会检查环境版本。

安装：
1. 上传 php.do 整个目录到网站。
2. 修改 config.php 的数据库信息。
3. 访问 install/install.php 自动建表。
4. 默认管理员：admin
5. 默认密码：admin123
6. 登录后请尽快修改密码，测试完成后删除 install/install.php。

功能：
1. 用户注册、登录、退出。
2. 版块列表。
3. 发帖、回帖。
4. 搜索帖子。
5. 后台新增版块。
6. 管理员置顶、加精、删除帖子、删除回复。
7. 记录发帖和回帖 IP。
8. 预留 IP 封禁表 qf_bans。
8. 发帖支持上传 jpg、jpeg、png、gif、webp、zip、rar。
9. 首页左侧默认显示最新回复排序的帖子，右侧显示版块。
10. 回帖支持上传 jpg、jpeg、png、gif、webp、zip、rar。
11. 管理员可以编辑帖子，普通用户不能编辑帖子。
12. 后台支持站点设置：浏览器标题、站内名称、网站简介、备案代码、统计代码、附件大小、允许上传后缀、单 IP 每日注册次数。
13. 后台支持编辑现有版块名称、简介和排序。
14. 后台支持广告位置：顶部、右侧板块上方、底部，可上传图片、设置宽高、链接和开关。
15. 后台版块支持一次性保存全部修改，并可删除版块。
16. 后台支持按版块设置主题分类开关和分类内容，发帖时按所选版块显示分类并显示在标题前。
17. 后台支持开启高级图片验证码，注册、发帖、回帖可防刷，并可设置回帖满多少次后免验证码。
18. 后台支持配置 S3/R2 对象存储，开启后图片/附件会上传到 S3/R2 空间。

升级说明：
如果你已经安装过旧版本，不需要重装数据库。
上传覆盖文件后，访问一次 install/upgrade.php 创建附件表。
新版还会创建 qf_settings 设置表。
升级成功后建议删除 install/upgrade.php。

说明：
这是第一版轻量论坛程序，界面清爽简约，适合先上传测试。
如果你测试满意，可以继续加：用户中心、头像上传、编辑帖子、验证码、附件图片、伪静态、Sitemap、后台封禁界面等。
